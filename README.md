## Installation

**Clone the source:**
```bash
git clone git@github.com:proxystore/aeris
```

**Create a virtual environment of your choosing:**
```bash
python -m venv venv
. venv/bin/activate
```

**Install the package & dependencies:**
```bash
pip install -e .  # -e for editable mode
```
